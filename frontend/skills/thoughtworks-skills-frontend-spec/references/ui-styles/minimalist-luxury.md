# 极简轻奢风格指南

## 设计调性

以留白和克制为核心美学，通过精选用色和细腻排版传递高级感。

## 色彩系统

| 角色 | 色值 | 说明 |
|------|------|------|
| 主色 | `#1B2A4A` | 深靛蓝，用于标题、按钮、强调元素 |
| 辅色 | `#C9A96E` | 香槟金，用于图标高亮、分隔线、badge |
| 背景色 | `#FAFAFA` | 近白灰，主体背景 |
| 卡片背景 | `#FFFFFF` | 纯白，卡片和面板 |
| 正文色 | `#333333` | 深灰，正文文字 |
| 辅助文字 | `#999999` | 中灰，次要文字、placeholder |
| 成功色 | `#52C41A` | 语义色 — 成功 |
| 警告色 | `#FAAD14` | 语义色 — 警告 |
| 错误色 | `#FF4D4F` | 语义色 — 错误 |
| 信息色 | `#1890FF` | 语义色 — 提示 |

## 设计 Token

| Token | 值 | 说明 |
|-------|-----|------|
| `--radius-sm` | `4px` | 小元素圆角（tag、badge） |
| `--radius-md` | `8px` | 中等元素圆角（按钮、输入框） |
| `--radius-lg` | `12px` | 大元素圆角（卡片、弹窗） |
| `--shadow-sm` | `0 1px 3px rgba(0,0,0,0.06)` | 微阴影 |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.08)` | 卡片阴影 |
| `--shadow-lg` | `0 8px 24px rgba(0,0,0,0.12)` | 浮层阴影 |
| `--spacing-unit` | `8px` | 间距基准单位（所有间距为 8 的倍数） |
| `--content-max-width` | `1200px` | 内容区最大宽度 |

## 组件风格指引

### 按钮

- 主按钮：`#1B2A4A` 实心填充，白色文字，hover 时亮度提升 10%
- 次要按钮：白色底 + `#1B2A4A` 边框 1px，hover 时填充 `#F5F5F5`
- 文字按钮：无背景无边框，`#1B2A4A` 文字，hover 时下划线
- 高度统一 40px，内边距 `16px 24px`，圆角 `--radius-md`

### 卡片

- 纯白背景，无边框，使用 `--shadow-md` 投影
- 内边距 `24px`，圆角 `--radius-lg`
- 标题与内容间距 `16px`

### 表格

- 表头背景 `#F7F8FA`，字重 500，字号 13px
- 行高 56px，行底部 `1px solid #F0F0F0` 分隔
- hover 行背景 `#FAFBFC`
- 无竖向分隔线

### 表单

- 输入框高度 40px，边框 `1px solid #E8E8E8`，圆角 `--radius-md`
- focus 状态：边框变为主色 `#1B2A4A`，外发光 `0 0 0 2px rgba(27,42,74,0.1)`
- label 在输入框上方，字号 13px，字重 500，颜色 `#666666`
- 必填标记用辅色 `#C9A96E` 星号

### 导航

- 顶部导航栏高度 64px，白色背景 + `--shadow-sm` 底部阴影
- 侧边导航宽度 240px，白色背景，选中项左侧 3px 主色竖线指示
- 导航项高度 48px，选中状态文字变为主色

### 空状态

- 居中排列：浅灰线条插图 + 一行灰色说明文字 + 可选操作按钮
- 插图使用线条风格，颜色 `#D9D9D9`
